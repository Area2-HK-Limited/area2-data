## 第一章：系統概覽

### 1.1 系統介紹

香港創基電子科技有限公司（Chinabase）網站內容管理系統（CMS）是一套基於 Quasar Framework 開發的現代化後台管理平台，用於管理公司官方網站的所有內容，包括最新消息、產品資訊、人才招聘、企業里程碑等。

### 1.2 系統功能架構

本系統包含以下主要功能模組：

| 模組 | 網址 | 功能說明 |
|------|------|---------|
| 最新消息 | /cms/News | 管理公司最新動態及公告 |
| 誠聘英才 | /cms/Recruitment | 發佈及管理招聘職位 |
| 發展里程 | /cms/Developing | 記錄公司發展歷史里程碑 |
| 橫額圖片 | /cms/BannerImage | 管理網站首頁橫額圖片/影片 |
| 專利證書 | /cms/Certificate | 展示公司資質認證 |
| 產品分類 | /cms/Category | 管理產品類別架構 |
| 產品 | /cms/Product | 管理產品規格參數 |
| 產品文件分類 | /cms/DocCategory | 管理產品技術文件分類 |
| 產品文件 | /cms/DocSubCategory | 管理產品技術文件 |
| 電郵訂閱 | /cms/Subscriber | 管理電子報訂閱者 |
| 產品查詢 | /cms/ProductInquiry | 查看客戶產品查詢記錄 |
| 客戶反饋 | /cms/Enquiry | 查看客戶聯絡表單記錄 |
| 翻譯 | /cms/Translate | 管理系統介面文字翻譯 |
| 檔案系統 | /cms/System/fs | 設定檔案上傳路徑 |
| 文件管理器 | /cms/FileManager | 瀏覽及管理伺服器檔案 |

### 1.3 多語言支援

本系統支援三種語言版本的內容管理：
- **[TC]** 繁體中文
- **[SC]** 簡體中文
- **[EN]** 英文

所有內容欄位均設有三個語言版本，管理員可同步填寫各語言版本內容，系統會自動按訪客所選語言顯示對應內容。

### 1.4 系統需求

- **瀏覽器：** 建議使用 Google Chrome、Microsoft Edge 或 Mozilla Firefox 最新版本
- **螢幕解像度：** 建議 1280×768 或以上
- **網絡連線：** 需要穩定的網絡連線

## 第二章：登入系統

### 2.1 功能說明

管理員需要通過登入頁面驗證身份，方可進入 CMS 後台管理系統。

### 2.2 登入步驟

**步驟一：** 開啟瀏覽器，前往 CMS 系統網址：  
`https://chinabase.hlhk.net/cms`

> 📷 *截圖參考：screenshots/chinabase/01_login.png*

系統會自動顯示登入頁面，介面包含：
- **用戶名稱欄位**（顯示人物圖示）
- **密碼欄位**（顯示鎖頭圖示）
- **Login 登入按鈕**
- **Forget password 忘記密碼連結**
- **指紋驗證按鈕**（如設備支援）

**步驟二：** 在用戶名稱欄位輸入您的帳號名稱  
**步驟三：** 在密碼欄位輸入對應密碼  
**步驟四：** 點擊 **「Login」** 按鈕

系統驗證成功後，會自動跳轉至 CMS 後台主頁（產品管理頁面）。

> 📷 *截圖參考：screenshots/chinabase/02_dashboard.png*

### 2.3 系統介面說明

登入後，CMS 介面分為以下幾個區域：

**頂部導航欄**
- 🍔 選單展開/收起按鈕
- ⛶ 全螢幕切換按鈕
- 🌐 語言切換按鈕
- 👤 用戶帳號按鈕（顯示目前登入帳號）

**左側導航選單**
- **設定**（系統設定）
  - 翻譯
  - 檔案系統
- **電郵訂閱**
- **產品查詢**
- **客戶反饋**
- **網頁更新**（網頁內容管理）
  - 最新消息
  - 誠聘英才
  - 發展里程
  - 橫額圖片
  - 專利證書
- **產品**（產品管理）
  - 產品分類
  - 產品
  - 產品文件分類
  - 產品文件
- **文件管理器**

**頁面底部資訊欄**
- 目前伺服器時間
- 顯示設定（顏色主題、選單模式等）

### 2.4 注意事項

- 請妥善保管您的登入帳號及密碼，切勿與他人共享
- 如使用公共電腦，完成操作後請務必登出系統
- 密碼區分大小寫，請確認 Caps Lock 狀態
- 如連續多次輸入錯誤密碼，帳號可能被暫時鎖定，請聯絡系統管理員處理

## 第三章：最新消息管理

### 3.1 功能說明

「最新消息」模組用於管理公司官方網站的最新動態及公告內容。每則消息支援圖片、三語標題及三語正文內容。

**系統路徑：** `/cms/News`

### 3.2 查看消息列表

進入最新消息頁面後，系統顯示消息列表，包含以下欄位：

| 欄位 | 說明 |
|------|------|
| News ID | 消息的唯一識別碼 |
| Image 01 | 消息封面圖片縮覽圖 |
| 標題[繁] | 繁體中文標題 |
| 標題[簡] | 簡體中文標題 |
| 標題[英] | 英文標題 |
| 建立日期 | 記錄建立的日期時間 |
| 狀態 | 顯示 ✓（啟用）或空白（停用） |

> 📷 *截圖參考：screenshots/chinabase/03_news.png*

列表支援以下功能：
- **搜尋**（🔍 按鈕）：按標題搜尋消息
- **排序**：點擊欄位標題可按升序/降序排序
- **分頁**：每頁顯示 20 條記錄（可調整），支援首頁/上頁/下頁/末頁切換
- **日期篩選**（📅 按鈕）：按日期範圍篩選

### 3.3 新增消息

**步驟一：** 點擊列表上方的 **「+ 新增」** 按鈕

系統彈出新增消息對話框，包含以下欄位：

> 📷 *截圖參考：screenshots/chinabase/news_form.png*

| 欄位 | 說明 | 是否必填 |
|------|------|---------|
| 圖片 01 | 消息封面圖片（點擊資料夾圖示上傳） | 選填 |
| Title [TC] | 繁體中文標題 | 選填 |
| Title [SC] | 簡體中文標題（可點擊翻譯按鈕自動翻譯） | 選填 |
| Title [EN] | 英文標題 | **必填** |
| Content [TC] | 繁體中文正文（富文本編輯器） | 選填 |
| Content [SC] | 簡體中文正文（富文本編輯器） | 選填 |
| Content [EN] | 英文正文（富文本編輯器） | 選填 |
| 狀態 | Active（啟用）或其他狀態 | 必填 |

**步驟二：** 點擊圖片欄位的 📁 資料夾圖示，選擇並上傳封面圖片

**步驟三：** 填寫各語言標題及內容

**步驟四：** 設定狀態（Active = 在前台顯示）

**步驟五：** 點擊 **「✓ 提交」** 按鈕儲存

### 3.4 修改消息

**步驟一：** 在消息列表中，找到要修改的記錄  
**步驟二：** 點擊該行的 ✏️ **編輯圖示**  
**步驟三：** 在彈出的編輯對話框中修改相應欄位  
**步驟四：** 點擊 **「✓ 提交」** 儲存變更

### 3.5 刪除消息

**步驟一：** 在消息列表中，找到要刪除的記錄  
**步驟二：** 點擊該行的 🗑️ **刪除圖示**  
**步驟三：** 系統彈出確認對話框，確認後點擊確認刪除

### 3.6 注意事項

- 圖片檔案名稱**請勿使用「.」（點）或空格**，可使用「-」或「_」符號
- 英文標題（Title [EN]）為必填欄位
- 消息設為「Active」狀態後，才會在前台網站顯示
- 已刪除的記錄無法復原，請謹慎操作

## 第四章：誠聘英才管理

### 4.1 功能說明

「誠聘英才」模組用於管理公司招聘職位資訊，支援三語職位名稱、部門、工作地點及有效時間等資訊。

**系統路徑：** `/cms/Recruitment`

### 4.2 查看職位列表

職位列表包含以下欄位：

| 欄位 | 說明 |
|------|------|
| Recruitment ID | 職位記錄唯一識別碼 |
| 職位名稱[繁/簡/英] | 三語職位名稱 |
| 招聘部門[繁/簡/英] | 三語招聘部門 |
| working_place_tc/sc/en | 三語工作地點 |
| 有效時間[繁/簡/英] | 三語職位有效期限 |
| 電郵 | 應聘電郵地址 |
| 建立日期 | 記錄建立日期 |
| 狀態 | 啟用/停用狀態 |

> 📷 *截圖參考：screenshots/chinabase/04_recruitment.png*

### 4.3 新增職位

**步驟一：** 點擊 **「+ 新增」** 按鈕

系統彈出新增職位對話框：

> 📷 *截圖參考：screenshots/chinabase/recruitment_form.png*

| 欄位 | 說明 |
|------|------|
| Position [TC/SC/EN] | 三語職位名稱 |
| Department [TC/SC/EN] | 三語招聘部門名稱 |
| Working Place [TC/SC/EN] | 三語工作地點 |
| Valid Time [TC/SC/EN] | 三語職位有效期限描述 |
| 電郵 | 應聘申請電郵地址 |
| 職位描述（富文本 TC/SC/EN） | 三語職位詳細說明 |
| 狀態 | Active / 停用 |

**步驟二：** 填寫職位資訊

**步驟三：** 在富文本編輯器中輸入職位詳細描述（支援格式化文字）

**步驟四：** 點擊 **「✓ 提交」** 儲存

### 4.4 注意事項

- 職位設為「Active」狀態才會在前台顯示
- 招聘截止後請將職位狀態改為停用，或直接刪除記錄
- 電郵地址請確認格式正確，供求職者投遞履歷使用

## 第六章：橫額圖片管理

### 6.1 功能說明

「橫額圖片」模組管理網站首頁的橫額（Banner）展示，支援靜態圖片及嵌入影片兩種格式。

**系統路徑：** `/cms/BannerImage`

### 6.2 查看橫額列表

橫額列表包含以下欄位：

| 欄位 | 說明 |
|------|------|
| Bannerimage ID | 記錄唯一識別碼 |
| 圖片 01 | 橫額圖片縮覽圖 |
| 影片嵌入 | 影片嵌入代碼（如有） |
| 是否影片 | 標示此橫額是否為影片 |
| 建立日期 | 記錄建立日期 |
| 狀態 | 啟用/停用 |

> 📷 *截圖參考：screenshots/chinabase/06_banner.png*

### 6.3 新增橫額

**步驟一：** 點擊 **「+ 新增」** 按鈕

> 📷 *截圖參考：screenshots/chinabase/banner_form.png*

| 欄位 | 說明 |
|------|------|
| Image 01 | 橫額圖片（點擊 📁 上傳圖片或影片檔案） |
| Is Video | 勾選此項表示此橫額為影片形式 |
| 狀態 | Active（啟用顯示於前台） |

**步驟二：** 點擊 📁 資料夾圖示選擇橫額圖片或影片檔案  
**步驟三：** 如為影片橫額，勾選「Is Video」  
**步驟四：** 設定狀態為「Active」  
**步驟五：** 點擊 **「✓ 提交」** 儲存

### 6.4 注意事項

- 橫額圖片建議尺寸請諮詢前端開發人員
- 圖片檔案名稱不可包含「.」（點）或空格
- 多個橫額圖片啟用時，將以輪播形式顯示
- 如使用影片橫額，需勾選「Is Video」以確保正確顯示

## 附錄：常見問題

### Q1：登入時提示帳號或密碼錯誤，應如何處理？

**A：** 請確認以下幾點：
1. 帳號及密碼是否正確（注意大小寫）
2. 鍵盤 Caps Lock 是否已關閉
3. 如確認輸入正確但仍無法登入，請聯絡系統管理員重設密碼

---

### Q2：上傳圖片後，前台網站沒有更新顯示，原因是什麼？

**A：** 可能的原因：
1. 圖片狀態未設為「Active」，請到對應模組將記錄狀態改為 Active
2. 瀏覽器快取問題，嘗試按 Ctrl+F5 強制重新載入前台頁面
3. 圖片上傳時出現錯誤，請重新上傳

---

### Q3：上傳檔案時提示錯誤，如何解決？

**A：** 常見原因及解決方法：
1. **檔案名稱含有特殊字符**：重命名檔案，移除「.」（點）、空格等字符，可使用「-」或「_」替代
2. **檔案太大**：壓縮圖片或文件至合理大小（建議圖片 < 5MB，PDF < 10MB）
3. **網絡超時**：確認網絡連線穩定後重試

---

### Q4：如何同時更新多個產品資料？

**A：** 使用「產品」模組的**「⬆️ 匯入產品列表」**功能，可通過上傳 Excel 文件批量更新產品資料。請按照系統指定的格式準備文件。

---

### Q5：如何刪除不需要的圖片或文件？

**A：** 
1. 先到相關模組（如「最新消息」）將使用該圖片的記錄刪除或更換圖片
2. 再到「文件管理器」（/cms/FileManager）找到並刪除舊檔案

---

### Q6：修改翻譯後沒有即時顯示，怎麼辦？

**A：** 嘗試重新整理瀏覽器頁面（F5 或 Ctrl+R），或清除瀏覽器快取後重新登入。

---

### Q7：如何查看客戶提交的產品查詢或反饋？

**A：** 
- **產品查詢**：進入「產品查詢」(/cms/ProductInquiry) 模組查看
- **客戶反饋**：進入「客戶反饋」(/cms/Enquiry) 模組查看

點擊記錄行可展開查看完整詳情。這兩個模組均為唯讀，需另外通過電郵聯絡客戶回覆。

---

### Q8：如何管理「發展里程」的顯示順序？

**A：** 發展里程記錄會按「發展日期」排序顯示，日期越近排越後。要調整顯示順序，可修改對應記錄的發展日期。

---

### Q9：新增產品時，技術規格欄位非常多，應該如何填寫？

**A：** 產品表單包含多種電子元件類型的規格欄位，只需填寫對應產品類型的相關欄位即可：
- **稽納二極管**：填寫 Vz Min/Nom/Max、Vz Izt Ma 等
- **三極管**：填寫 Ic、Vcbo/Vceo、hFE、Ft 等
- **MOSFET**：填寫 Id、Vdss、vgs 等
- **LED**：填寫光強（mcd）、波長、順向電壓等

其他類型欄位可留空，系統不強制要求所有欄位都填寫。

---

### Q10：如何確保前台網站正確顯示最新內容？
