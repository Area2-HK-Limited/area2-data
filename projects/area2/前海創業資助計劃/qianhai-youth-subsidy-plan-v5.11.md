### 每人淨收益

| 學歷 | 總資助 | - 每人成本 | = 每人淨收益 |
|------|--------|-----------|-------------|
| **學士** | ¥128,000 | -¥75,032 | **¥52,968** |
| **高級文憑** | ¥92,000 | -¥74,179 | **¥17,821** |

> **分配邏輯（v5.10）：**
> 1. Agent Fee 按資助額比例分擔：學士 ¥3,032/人，高級文憑 ¥2,179/人
> 2. 每人總成本 = 社保 + 租枱 + Agent Fee（學士 ¥75,032，高級文憑 ¥74,179）
> 3. 每人淨收益 = 資助 - 每人總成本
> 4. 總淨收益 = ¥696,000 - ¥432,000 - ¥16,488 = **¥247,512**
> 5. 利潤分攤：員工 20% = **¥49,502**，Area2 80% = **¥198,010**

> **分配邏輯（v5.10）：**
> - 總資助：¥696,000（4人學士 + 2人高級文憑）
> - 總成本：¥432,000（社保+租枱）+ ¥16,488（Agent Fee）= **¥448,488**
> - 總淨收益：¥696,000 - ¥448,488 = **¥247,512**
> - Agent Fee 按資助額比例分擔：學士 ¥3,032/人，高級文憑 ¥2,179/人

> ✅ **全員均為正收益！** Agent Fee 按資助額比例分擔（學士 ¥3,032，高級文憑 ¥2,179），而非平均分擔。
>
> **利潤分成說明：** 員工 20% 利潤池共 ¥49,502，按每人淨收益比例分配：
> - 學士（4人）：¥52,968 ÷ ¥247,512 × ¥49,502 = **¥10,594/人**
> - 高級文憑（2人）：¥17,821 ÷ ¥247,512 × ¥49,502 = **¥3,564/人**

## ⚠️ 社保繳費規則說明（重要更正）

> **背景：** 網上有錯誤說法：「月薪5000以下不用報稅，所以社保交少啲」。本文件特別澄清。

### 🔑 社保 vs 個稅：兩件完全不相關的事

| | 社會保險（社保） | 個人所得稅 |
|---|---|---|
| **法律依據** | 《社會保險法》強制繳納 | 《個人所得稅法》 |
| **5000元** | ❌ 完全無關 | ✅ 起征點（月薪5000以下免個稅）|
| **繳費規則** | 按繳費基數比例計算 | 超過起征點按累進稅率計算 |
| **是否強制** | **強制繳納**，不由工資高低決定 | 超過5000元才需繳 |

### 📊 深圳社保繳費基數（2024-2025）

| 險種 | 下限（元/月） | 上限（元/月） |
|---|---|---|
| **養老保險** | ¥4,492 | ¥27,501 |
| **醫療保險** | ¥6,733 | ¥33,666 |
| **失業保險** | ¥4,492 | ¥27,501 |

**繳費規則：**
- 月薪 **< 下限** → 按**下限**作為基數繳費
- 月薪 **在上下限之間** → 按**實際月薪**作為基數繳費
- 月薪 **> 上限** → 按**上限**作為基數繳費

### 💰 以月薪 ¥5,000 為例（HD 高級文憑）

假設員工月薪 ¥5,000：

| 險種 | 基數選取 | 員工比例 | 每月扣除 |
|------|---------|---------|---------|
| 養老保險 | ¥5,000（> 下限¥4,492，按實際） | 8% | ¥400 |
| 醫療保險 | ¥6,733（< 下限，按下限） | 2% | ¥135 |
| 失業保險 | ¥4,492（< 下限，按下限） | 0.2% | ¥9 |
| **合計** | | | **¥544/月** |

**實際到手：** ¥5,000 - ¥544 = **¥4,456**

### 📊 對比：月薪 ¥10,000（學士）

| 險種 | 基數選取 | 員工比例 | 每月扣除 |
|------|---------|---------|---------|
| 養老保險 | ¥10,000（> 下限，按實際） | 8% | ¥800 |
| 醫療保險 | ¥10,000（> 下限¥6,733，按實際） | 2% | ¥200 |
| 失業保險 | ¥10,000（> 下限，按實際） | 0.2% | ¥20 |
| **合計** | | | **¥1,020/月** |

**實際到手：** ¥10,000 - ¥1,020 = **¥8,980**

### ✅ 結論

1. **HD 人工低確實社保交少啲** — 但衹是因為部分險種按下限計算
2. **幅度有限** — 月薪¥5,000比¥10,000少交¥476社保，但學士實際到手仍多¥4,524
3. **本 plan 假設社保基數 ¥9,000/月** — 高於各險種下限，按實際薪資計算，無合規風險
4. **Agent Fee 數字已更新：** 學士 ¥3,032/人，高級文憑 ¥2,179/人（按資助額比例分擔）
