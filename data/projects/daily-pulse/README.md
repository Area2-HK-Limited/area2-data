# Daily Pulse 🌅

AI-powered morning briefing dashboard for busy professionals.

## Features

- 📅 **Calendar Integration** - iCloud Calendar sync with automatic categorization
- 🌤️ **Weather Forecast** - Hong Kong Observatory data for your locations
- 📈 **Market Data** - SPX and stock tracking via yFinance API
- 🤖 **AI Briefing** - Daily summary generated by GPT-4
- 📱 **Mobile-First** - Responsive design for all devices

## Quick Start

### Prerequisites

- Docker & Docker Compose
- Node.js 20+ (for local development)
- OpenAI API key (for AI briefings)

### Setup

1. Clone and configure:
   ```bash
   cd daily-pulse
   cp .env.example .env
   # Edit .env with your API keys
   ```

2. Start with Docker Compose:
   ```bash
   docker compose up -d
   ```

3. Open http://localhost:3000

### Local Development

```bash
cd frontend
npm install
npm run dev
```

## Configuration

### Environment Variables

| Variable | Description | Required |
|----------|-------------|----------|
| `POSTGRES_PASSWORD` | Database password | Yes |
| `JWT_SECRET` | PostgREST JWT secret | Yes |
| `YFINANCE_API` | yFinance API endpoint | No (has default) |
| `YFINANCE_KEY` | yFinance API key | Yes |
| `OPENAI_API_KEY` | OpenAI API key for briefings | Yes |
| `ICAL_URL` | iCloud Calendar webcal URL | No |

### Calendar Setup

1. Go to iCloud Calendar → Share Calendar
2. Copy the public URL (webcal://...)
3. Add to `.env` as `ICAL_URL`

## Tech Stack

- **Frontend:** Nuxt 3 + Nuxt UI Pro
- **Database:** PostgreSQL 16
- **API:** PostgREST
- **Weather:** HK Observatory API
- **Market Data:** yFinance API
- **AI:** OpenAI GPT-4o-mini

## API Endpoints

### Frontend Server Routes

- `GET /api/weather?locations=荃灣,旺角` - Weather data
- `GET /api/calendar/events?url=...&days=7` - Calendar events
- `POST /api/briefing` - Generate AI briefing

### PostgREST (port 3001)

- `GET /users` - User list
- `GET /calendar_events` - Cached events
- `GET /market_data` - Market data cache
- `GET /briefings` - Historical briefings
