## 1. 客戶服務 / 業務團隊日常工作地圖

### 1.1 核心職責板塊

| 板塊 | 日常任務 | 涉及系統/人 |
|------|----------|-------------|
| **接新訂單** | 接收客戶brief、評估可行性、內部協調資源 | WhatsApp / 電話 / Email |
| **客戶溝通** | 確認需求細節、傳遞想法、解答疑問 | WhatsApp、Email、電話 |
| **項目進度更新** | 監察執行進度、主動通知客戶、處理臨時變更 | WhatsApp、Excel、內部群組 |
| **投訴處理** | 記錄投訴、分配跟進、閉環確認 | WhatsApp、Email、Word |
| **報價準備** | 整理客戶歷史、估算造價、輸出報價單 | Excel、Email |

### 1.2 現有痛點（未應用 SecrexAI 前）

```
痛點 A：資訊碎片化
→ 客戶問「我哋個項目進度點？」→ CS 要逐個 WhatsApp 群 / 電話確認
→ 由查詢到回覆，中間起碼 15-30 分鐘

痛點 B：被動式服務
→ 客戶不主動問 = CS 不知道要通知
→ 客戶臨時變更（例如加印數量）→ 未能及時通知設計/印刷環節
→ 結果：客戶投訴「你哋完全冇主動通知我」

痛點 C：跨部門協調低效
→ 報價需要問設計部、排期需要問印刷部、生產需要問物流
→ 來回來回，報個價要 1-2 天

痛點 D：投訴紀錄靠人手
→ WhatsApp 對話Screenshot → 手動貼去 Word
→ 無結構化，日後查不到
→ 閉環與否全靠個人記憶
```

### 1.3 Invoice Ninja Project Module 上線後，仍未解決的問題

| Invoice Ninja 做到的 | Invoice Ninja 做不到的 |
|----------------------|--------------------------|
| 項目收支有系統記錄 | 客戶自己無法查進度（未有 portal） |
| 發票 / 付款狀態可追蹤 | CS 無法快速向客戶總結「今日狀況」 |
| 項目利潤可見 | 跨 WhatsApp 的溝通紀錄無法關聯到項目 |
| 報價可模板化 | 投訴的跟進狀態無自動提醒 |

**核心缺口：Invoice Ninja 是內部工具，客戶看不見；CS 還是需要花大量時間做「資訊翻譯」——把系統資料用人話講給客戶。**

## 2. SecrexAI 可以扮演的角色（CS 視角）

### 角色定位：**客戶服務的「超能助理」**

SecrexAI 不是取代 CS，而是讓 CS 變成「WhatsApp 就能驅動的作戰中心」。

### 具體場景

#### 場景 A：客戶查進度 → SecrexAI 即時回覆

```
客戶發 WhatsApp：「我想知我哋嗰個恒基示範單位嘅項目進度」
         ↓
CS 同事 forwarded 給 SecrexAI（或設定自動路由）
         ↓
SecrexAI 自動回覆：
「恒基A盤示範單位項目狀況（截至今天 10:00）：

✅ 設計確認：已確認（4月10日）
⏳ 印刷：進行中（預計4月16日完成）
⏳ 安裝：排期中（客戶指定日期後通知）

款項狀態：已付50%，尾款4月20日到期
客戶聯絡人：陳小姐 / 旺角門店

如需更多細節，請回覆『詳細』或『聯絡負責同事』。」
         ↓
CS 轉發給客人，耗時 < 1 分鐘
```

#### 場景 B：主動推送進度更新（避免客戶投訴「冇通知我」）

```
觸發條件：Invoice Ninja 狀態變更（例如印刷狀態改為「完成」）
         ↓
SecrexAI 自動生成 WhatsApp 訊息模板：
「【洗樓王項目更新】恒基A盤示範單位

📦 印刷已完成！
下一步：安裝（排期 4月18日，請確認時間）

如有任何問題，請回覆此訊息或致電我們。」
         ↓
CS 確認後一鍵發送 / 或設定自動發送（管理層審批模式）
```

#### 場景 C：投訴記錄 + 智能跟進

```
客戶投訴：「今日安裝師傅遲到兩粒鐘都冇通知我！」

CS 轉發給 SecrexAI，指令：「記錄投訴，相關項目：恒基A盤，安裝延誤，
涉及物流部門，需要4月16日前回覆客戶。」

         ↓
SecrexAI 自動：
1. 在相應項目筆記欄寫入投訴記錄（帶時間戳）
2. 創建跟進任務：「回覆恒基A盤客戶：安裝延誤 — 截止4月16日」
3. 通知相關同事（如有配置）
4. 等待 CS 輸入回覆方案後，協助草擬回覆訊息
```

#### 場景 D：報價準備加速

```
CS 指令：「幫我準備恒基今次嘅新項目報價，50個舖面巡邏派發，
為期兩周，請根據過往同類項目估算。」

         ↓
SecrexAI 輸出：
- 過往恒基類似項目平均造價參考
- 系統自動估算工時 / 成本
- 初步報價草稿（供 CS 審核後發出）
- 預計利潤率提示
```
