## 📋 文件概述

本文件為 **Area2 (HK) Limited** 制作的內部規劃方案，核心目的是評估申請「深圳前海港澳青年創業相關資助」的**財務可行性**，包括：

> ⚠️ 本文件包含敏感的財務假設，請勿對外轉發。
> ⚠️ **核心理念：** 員工是創業的重要夥伴，Area2 將政府補貼帶來的收益分享予員工。

## 💰 財務總結

### 總數檢查

| 項目 | 金額 |
|------|------|
| 6人總資助收入 | ¥660,000 |
| 總成本（社保+租枱） | ¥432,000 |
| Agent Fee | HK$18,888 |
| **總成本合計** | **¥432,000 + HK$18,888（≈¥16,488）** |
| **總淨收益** | **¥211,512**（資助 - 社保 - 租枱 - Agent Fee）|

### 利潤分享（Area2 80% / 員工 20%）

| 分配方 | 金額 | 說明 |
|----------|------|------|
| **Area2（80%）** | **¥169,210** | 從總淨收益中提取 |
| **員工（20%）** | **¥42,302** | 從總淨收益中提取 |

### 每人淨收益

| 學歷 | 總資助 | - 每人成本 | = 每人淨收益 |
|------|--------|-----------|-------------|
| **學士** | ¥128,000 | -¥75,198 | **¥52,802** |
| **高級文憑** | ¥92,000 | -¥74,298 | **¥17,702** |

> **分配邏輯（v5.8）：**
> 1. Agent Fee 按資助額比例分擔：學士 ¥3,198/人，高級文憑 ¥2,298/人
> 2. 每人總成本 = 社保 + 租枱 + Agent Fee（學士 ¥75,198，高級文憑 ¥74,298）
> 3. 每人淨收益 = 資助 - 每人總成本
> 4. 總淨收益 = ¥660,000 - ¥432,000 - ¥16,488 = **¥211,512**
> 5. 利潤分攤：員工 20% = **¥42,302**，Area2 80% = **¥169,210**

> **分配邏輯（v5.8）：**
> - 總資助：¥660,000
> - 總成本：¥432,000（社保+租枱）+ ¥16,488（Agent Fee）= **¥448,488**
> - 總淨收益：¥660,000 - ¥448,488 = **¥211,512**
> - Agent Fee 按資助額比例分擔：學士 ¥3,198/人，高級文憑 ¥2,298/人

> ✅ **全員均為正收益！** Agent Fee 按資助額比例分擔（學士 ¥3,198，高級文憑 ¥2,298），而非平均分擔。

- 資助款項預計在申請後約 **6 個月**方可回收

- 如政府延遲 / 分批 / 少付，按實際情況重新計算，比例（20%）維持不變

| 風險類型 | 說明 | 應對策略 |
